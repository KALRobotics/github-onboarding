# Collaboration Tips

## Commit Messages
Good: "Add intake motor control"
Bad: "stuff" / "fixed it"

## Review Etiquette
**Author:** Keep PRs small, respond to feedback
**Reviewer:** Be kind, explain why

## Stay Synced
Pull often to avoid conflicts:
```bash
git pull origin main
```
