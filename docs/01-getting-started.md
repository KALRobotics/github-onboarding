# Getting Started

## 1. Create GitHub Account
1. Go to https://github.com
2. Sign up with recognizable username
3. Add profile photo

## 2. Get Added to Team
1. Tell a captain your username
2. Accept the org invite

## 3. Install Git
- **Mac:** Terminal → `git --version`
- **Windows:** https://git-scm.com

## 4. Configure Git
```bash
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
```

## 5. Install GitHub Desktop (Recommended!)
https://desktop.github.com — easier for beginners
