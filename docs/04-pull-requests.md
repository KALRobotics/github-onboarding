# Pull Requests (PRs)

PRs ask teammates to review and merge your code.

## Open a PR
1. Push your branch
2. Go to repo on GitHub
3. Click "Compare & pull request"
4. Add title, description, reviewer
5. Create!

## Good PR Description
```
## What
Adds intake motor control

## How to test
1. Deploy to robot
2. Press A button
3. Intake spins
```

## After Approval
1. Click "Merge"
2. Delete branch
