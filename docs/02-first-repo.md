# Your First Repository

## Clone (Download)
**GitHub Desktop:**
1. Click "Clone repository"
2. Find repo, click Clone

**Terminal:**
```bash
git clone https://github.com/KALRobotics/repo-name.git
cd repo-name
```

## Make Changes
1. Open files in editor
2. Make changes, save

## Commit (Save Snapshot)
**GitHub Desktop:** Write summary, click "Commit"

**Terminal:**
```bash
git add .
git commit -m "Add my name to roster"
```

## Push (Upload)
**GitHub Desktop:** Click "Push origin"

**Terminal:**
```bash
git push
```
