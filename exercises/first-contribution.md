# Exercise: First Contribution

## Goal
Make a real contribution!

## Steps
1. Clone this repo
2. Create branch: `practice/your-name`
3. Add your name to [contributors.md](contributors.md)
4. Commit: "Add [your name]"
5. Push
6. Open Pull Request
7. Request review
8. Merge!
